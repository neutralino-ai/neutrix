4.@-mentions in user input — inline vs attachdecided
CC
attaches file content as separate attachment blocks alongside the user message.
Decision
Alternative — inline expansion. neutrix is OpenAI-compat text + append-only, so @path in a typed message is expanded into the SAME user turn: the raw text followed by <file path="…">…</file> blocks (line ranges via @path#L10-20; a dir → a file listing; missing → a note). Done in _worker_loop after dequeue (so the queue panel shows the raw typed text). The expanded turn is what the LLM sees and the transcript shows (visibility parity). Recall surfaces the expanded text — acceptable for v1.2.0.
