# Agent Instructions

For any implementation task in this repository, agents must follow this workflow:

1. Write or update the PRD first.
2. Develop the code.
3. Run or present the user-based acceptance test, and confirm with the user
   that the result is acceptable and acceptance testing can pass.
4. Write or update the CHANGELOG.
5. Commit, tag, and push the changes.

Design and implementation must follow the SOLID and YAGNI principles.
